# UniEvents - College Event Registration System

A premium frontend-only accessible React application for college event registration.

## Features
- **Browse Events**: View detailed cards for upcoming campus events.
- **Registration**: Register with student details (validated).
- **Payment Simulation**: Secure-looking credit card payment modal (mock).
- **Confirmation**: Receive a digital ticket with transaction details.
- **Responsive Design**: Mobile-friendly layout with modern aesthetics.

## Tech Stack
- React (Hooks)
- Vanilla CSS (Variables, Flexbox, Grid)
- Lucide React (Icons)
- Vite (Build tool)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually `http://localhost:5173` or `5174`).
